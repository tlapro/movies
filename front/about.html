Acerca del Sitio

Estas son las tecnologías con las que se ha desarrollado esta página web.

JavaScript

Es un lenguaje de programación que se usa para dar interactividad a las páginas web, agregando eventos, actualizando contenido, entre otras funciones.

CSS

Es el lenguaje de estilos que se usa para diseñar y dar formato a una página web. Controla colores, fuentes, espaciados, distribución de los elementos, etc.

HTML

Es el lenguaje de marcado que estructura el contenido de las páginas web. Define elementos como párrafos, encabezados, imágenes, botones, etc.

Es un framework CSS que facilita el diseño de páginas web responsivas (que se adaptan a diferentes pantallas). Proporciona componentes y estilos predefinidos, como botones, tarjetas, menús, etc., y es compatible con JavaScript para mejorar la interacción.
